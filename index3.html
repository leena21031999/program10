SERVICE MODELS

    IaaS PaaS SaaS

IaaS:- Infrastructure as a service means the outsourcing of the infrastructure of the IT network storage and server from the third party provider

PaaS:- Platform as a service offers an environment on which developers can create and deploy application and do not necessory need to know how many processes or how much memory that application will be using

SaaS:- Software as a service provide over the internet and requires no prior installation there services can be used from anywhere
